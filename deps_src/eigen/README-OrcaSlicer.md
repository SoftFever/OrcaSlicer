# Eigen in OrcaSlicer.

> [!NOTE]
> This is not the complete eigen distribution! Only the files needed for
> compiling eigen into OrcaSlicer are included in the OrcaSlicer source
> distribution. The full eigen distribution can be found at;
>
> https://libeigen.gitlab.io/.

This directory contains parts of the eigen v5.0.1 (bc3b3987) source
distribution together with a minimal custom `CMakeLists.txt` file. In
particular only the following files from the full eigen distribution are
included;

* COPYING.README
* README.md
* Eigen/*
* unsupported/Eigen/*
